# 📂 Documentation & Visual Assets

This directory contains visual documentation and supporting materials for the **Multi-Property GA4 Analytics** project. It provides a quick visual overview of the final business intelligence outputs, the underlying data model, and the ETL architecture.

## 📊 Final Reporting & UX

* **`report_preview_first_page.png`**: A static snapshot of the Executive Summary dashboard. This view highlights dynamic YoY KPI thresholds (▲/▼), and the ability to toggle between core metrics (e.g., Active Users, Engagement Time) utilizing Power BI Bookmarks while preserving custom Y-axis formatting.
![report first page](report_preview_first_page.png)

* **`report_preview_second_page.png`**: A granular deep-dive view demonstrating domain-level performance comparison and detailed channel acquisition breakdowns.
![report second page](report_preview_second_page.png)

> 📁 *The complete `.pbix` source file and DAX measure exports are stored in the [power_bi](../power_bi/) directory. *You can explore the fully interactive version of this dashboard online at **[Maven Showcase - Link Here](https://mavenshowcase.com/project/56439)**.*
